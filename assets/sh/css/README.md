# SH CSS Modules

Load `sh-core.css` first, then include the feature bundles that match the storefront areas you need.

| File | Purpose |
| --- | --- |
| `sh-core.css` | Base typography import, color variables, button defaults, and shared form radii. |
| `sh-layout-utilities.css` | Layout helpers, body margin reset, trustbadge toggle, and icon alignment fixes. |
| `sh-product-listing.css` | Product card pricing, UVP handling, and bundle badge overrides. |
| `sh-product-detail.css` | Detail page purchase panel, variant chooser, and availability note styling. |
| `sh-free-shipping.css` | Styles for the SH free-shipping progress banner. |
| `sh-checkout-cart.css` | Checkout/cart improvements covering fast checkout, quantity controls, shipping icons, and totals emphasis. |
| `sh-forms.css` | Retouren form layout adjustments. |
| `sh-wishlist.css` | Wishlist button treatment and icon visibility rules. |
